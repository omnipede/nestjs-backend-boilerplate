# Nest.js backend server boilerplate

## Description

* [Nest](https://github.com/nestjs/nest) framework TypeScript 을 사용하여 만든 백엔드 API 서버 boilerplate
* 매번 프로젝트 세팅하기 번거로워 만들게 됨

## Installation

```bash
$ npm install
```
